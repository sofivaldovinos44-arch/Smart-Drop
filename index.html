Smart Drop 💧

Cada gota cuenta

En México, estados como Chihuahua, Sonora y Nuevo León enfrentan sequías que reducen el agua disponible para el riego. Esto afecta tanto a pequeños cultivos como al cuidado de plantas en jardines.

Para ayudar a enfrentar este problema desarrollamos Smart Drop, un sistema de riego automatizado que optimiza el uso del agua y evita desperdicios mediante tecnología accesible.

Porque cuando el agua es limitada, cada gota cuenta… y Smart Drop está diseñado para ayudarte a cuidarla mientras tus plantas siguen creciendo.

Cómo hacer un Smart Drop

A continuación se muestra una guía básica para construir un sistema de riego automatizado Smart Drop utilizando componentes electrónicos simples. Este sistema permite regar plantas automáticamente y optimizar el uso del agua.

Primero se conectan los componentes principales: el Arduino, el sensor de humedad y la bomba de agua utilizando una protoboard y cables de conexión. El sensor se coloca en la tierra para medir la humedad del suelo y enviar información al sistema.

Cuando el sensor detecta que la tierra está seca, el Arduino activa la bomba de agua para comenzar el riego. De esta manera el sistema mantiene las plantas hidratadas automáticamente y evita el desperdicio de agua.

Información adicional

Smart Drop es un proyecto desarrollado con el objetivo de demostrar cómo la tecnología puede aplicarse para resolver problemas reales relacionados con el uso del agua. A través de componentes electrónicos accesibles y energía sustentable, este sistema propone una alternativa simple para automatizar el riego de plantas y pequeños cultivos.

Este tipo de soluciones pueden adaptarse a jardines domésticos, huertos escolares y pequeños espacios de cultivo, ayudando a mejorar el cuidado de las plantas y a reducir el desperdicio de agua.

El desarrollo de Smart Drop también busca fomentar la innovación, el aprendizaje tecnológico y la conciencia sobre la importancia de utilizar los recursos naturales de forma responsable.
